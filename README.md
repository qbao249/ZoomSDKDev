### About

This is cocoapod for Zoom IOS SDK. 

Pod includes all architectures.
(Device and Emulator)


Pod includes sdk list:
- MobileRTC.framework
- MobileRTCScreenShare.framework
- MobileRTCResources.bundle

### Usage
Update your Podfile:
```
pod 'ZoomSDK', '5.5.12511.0421'
```


### Versions

- 5.5.12511.0421
- 5.4.54802.0124.1
